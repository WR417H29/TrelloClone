# Trello Clone App

## To Run

### Frontend

-   cd web && npm start

### Backend

-   cd server
-   activate environment ( if in use )
-   python main.py

## Naming Conventions

-   Classes and Components -> Pascal Case (Upper Camel Case)
-   Functions and Variables -> Camel Case
-   File Names -> Camel Case
-   Braces open on end of line
